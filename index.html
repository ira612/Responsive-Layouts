One pricing section. Two ways to build it.

This project demonstrates how the same responsive pricing layout can be built using two different CSS layout systems — Flexbox and CSS Grid — while keeping the visuals and user experience identical.

Pick a version to explore

Both pages share the same HTML structure, content, and visual design — only the pricing card layout technique differs.

Version 1

Flexbox Layout

Uses display: flex, flex-wrap, justify-content, and gap to arrange the three pricing cards in a single flexible row.

  • File: flexbox.html + css/flexbox.css
  • One-dimensional layout model
  • Great for content-driven sizing
View Flexbox Version
Version 2

CSS Grid Layout

Uses display: grid, grid-template-columns, and grid-template-areas to place cards on an explicit two-dimensional grid.

  • File: grid.html + css/grid.css
  • Two-dimensional layout model
  • Great for precise, structural placement
